# 0. Overview
In this post, our aim is to create a VM runs a Mac os for your lab environment.

Please note that you need a mac os I mean you literaliy need a physical mac devices to run a VM which run another mac os..

I thought the mac os never have been provide a their installer like iso but somehow I have struggled to troubleshoot mac devices for some reason, I think I've been wrong!

So, this post will let you led how to create a VM runs mac os.

[Index]

1. Download UTM application
2. Download IPSW
3. Create VM
